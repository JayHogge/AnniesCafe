Course Evaluation


While taking this web development course I went into it with hopes of getting into JavaScript scripting and a more back-end focused curriculum. However, I have only ever designed websites using w3schools documentation and didn't dive into detail about front-end use so despite my early expectations not being met, I was pleased and find more value in the things I learned. Before this course I hadn't understood how to use the devtools, especially the mobile-viewer or how submission forms use GET and other protocols to send information to webservers. The mobile development portion of the class was particularly useful and I appreciate better knowing what terms like 'bootstrap' and what inline versus flexbox or box elements are.

I was asked for help by the owner of Annie's Cafe with revamping their website and together we discussed the aesthetic and atmosphere of their Cafe and how to best present that online. The website upgrade transformed from a damp sky blue color, to the modern dark gray, Gold, and whitesmoke colors. These were inspired as much of their interior uses warm tones, incorporates modern interior elements like grayrod walls and golden trim to create a high-end feel. The white and black colors contrast and compliment these other colors well and there wasn't another color that would match the palate well besides other shades of gray, so I hope it's not contrite or a drab on the whole project.

Had I allotted more time to this project I would have loved to improved the CSS layout of the site. The biggest flaw of the currently published Annies site is its poorly designed mobile site. The website was made with very limited web-dev tools that were provided by SpotOn. The owner isn't a designer and recognized that the mobile website needed a lot of work. The nav that uses the hamburger function especially shines through on my mobile draft, and had I more time I would've incorporated itsacheckmate with UberEats, DoorDash, and GrubHub hyperlinks. I also was hoping to make the hamburger menu slide across with a white-translucent overlay page instead of the current dropdown style.
